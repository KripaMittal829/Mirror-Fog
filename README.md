
# 🌫️ Mirror Fog 

Mirror Fog is an interactive foggy mirror effect that blends **gesture control**, **real‑time tracking**, and **audio detection** to create a playful, immersive experience.  
Users can wipe fog with their fingers, re‑fog the screen, or even trigger mist using breath motion.

---

## 🚀 Features
- ✋ **Realtime Finger Tracking** — pinch, swipe, and gestures detected via MediaPipe Hands.  
- 🌬 **Breath Detection** — fog responds to mic input using Web Audio API.  
- 🖼 **Fog Rendering** — Canvas 2D API draws and animates the mist layer.  
- 🔄 **Re‑fog & Wipe** — instantly clear or re‑apply fog for dynamic interaction.  
- 📷 **Camera Integration** — enables gesture recognition and immersive control.  

---

## 📂 Tech Stack
- **MediaPipe Hands** — realtime finger tracking and gesture recognition  
- **Canvas 2D API** — renders fog effects and animations  
- **Web Audio API** — detects breath via microphone input  
- **JavaScript (Vanilla)** — core logic and event handling  
- **HTML5 + CSS3** — structure and styling
